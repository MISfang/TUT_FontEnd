<h1>天理通小程序 后台管理系统前端</h1>

## 介绍

天理通小程序是一款专注于为天津理工大学校园圈师生提供便捷服务的小程序

本项目为该小程序后台管理系统的前端

#### 在线地址预览

http://47.94.128.16/#/login

#### 后端项目链接

https://github.com/MISfang/TUT_BackEnd

#### 技术栈

- Vue3 主框架
- vite2 打包构建工具
- Typescript 静态类型检查
- pinia 状态管理
- Vben[前端框架](https://vvbin.cn/doc-next/) 主要 UI 框架
- axios 请求处理
- dayjs 时间戳处理
- [图标库](https://iconify.design/)
- Echart-Vue 数据可视化
